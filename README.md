# NeuroCanvas Studio

A dementia-friendly digital arts platform for painting, drawing, calming music, and voice-led support. Built with Next.js, Supabase, and a focus on accessibility and clinical review.

## Features
- Layered painting and drawing canvas
- Large, accessible controls
- Music-assisted sessions
- Voice guidance
- Caregiver dashboard
- Supabase backend for session and artwork storage

## Getting Started
1. Clone the repo: `git clone https://github.com/chrsappiah-cloud/neurocanvas-studio.git`
2. Install dependencies: `npm install`
3. Create a `.env.local` file with your Supabase credentials
4. Run the app: `npm run dev`
